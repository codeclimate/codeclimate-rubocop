This cop makes sure that all variables use the configured style,
snake_case or camelCase, for their names.

### Example: EnforcedStyle: snake_case (default)
    # bad
    fooBar = 1

    # good
    foo_bar = 1

### Example: EnforcedStyle: camelCase
    # bad
    foo_bar = 1

    # good
    fooBar = 1
