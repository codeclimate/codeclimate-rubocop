This cop checks if empty lines exist around the arguments
of a method invocation.

### Example:
    # bad
    do_something(
      foo

    )

    process(bar,

            baz: qux,
            thud: fred)

    some_method(

      [1,2,3],
      x: y
    )

    # good
    do_something(
      foo
    )

    process(bar,
            baz: qux,
            thud: fred)

    some_method(
      [1,2,3],
      x: y
    )
